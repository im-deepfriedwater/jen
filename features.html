jen

jen is a strongly and statically-typed scripting language. Featuring inferred typing, multiple return values, and static scoping. Let's dive into the details!

Expressions

Literals: Expression Literals are made up of the same 8 data types mentioned above.

Function Calls: jen Functions are called in the form: functionName(parameters)

Logical NOT: Not expression, represented as a bang !, negates any logical expression before it

Arithmetic: Expressions such as add, subtract, multiply, divide, etc

Boolean: The typical AND, and OR are included, as well as the XOR expression

Ternary: jen features a ternary expression, formatted in the same fashion as JavaScript (Conditional ? if true : else)

Statements

Loops: jen features both for, and while loops, using similar syntax to Python

Function Declaration: jen functions are declared using both an annotation, and a signature

Declaration: variables in jen are declared and assigned using different syntax.

Assignment: variables are assigned using the = sign

Conditional: jen features both the aforementioned ternary operator, along with the standard if statement. Following similar syntax to Python

Return: naturally, jen allows functions to return a value, using the return keyword
