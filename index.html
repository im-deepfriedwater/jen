Say hello to your new best friend:

jen

Drawing inspiration from JavaScript, Python, Go, Typescript's sum type, and even Elm, jen provides a pleasant and happy programming experience for all your high level needs. jen's powerful type system provides expressiveness, but also watches your back to prevent head-scratching type errors. With conciseness, elegance, and functionality, jen works hard as a language so you don't have to!

jen is a strongly and statically-typed scripting language. Featuring inferred typing, multiple return values, and static scoping.
For more information about the grammar behind jen,
refer to jen's Ohm grammar to see how it all fits together

Operators:

  • Comparison Operators:

  • Equal: ==

  • Not Equal: !=

  • Less Than: <

  • Greater Than: >

  • Less Than or Equal: <=

  • Greater Than or Equal: >=

  • Logical Operators:

  • Logical AND &&

  • Logical OR ||

  • Logical NOT !

  • Logical XOR &!&

  • Mutative Operators:

  • Postfix/Prefix Incremement: ++

  • Postfix/Prefix Decriment: --

  • Assignment Operator: =

  • Arithmetic Operators:

  • Add: +

  • Subtract: -

  • Multiply: *

  • Divide: /

  • Integer Divide: //

  • Modulous: %

  • DivMod: /%

  • Power: ^

Data Types:

  • Boolean

  • Number

  • List

  • String

  • Record

  • Any

  • Void

  • Error

Built in Functions:

  • Square root: sqrt()

  • Pi: pi()

  • String lowercase: toLower()

  • String uppercase: toUpper()

  • Random number: random()

Expressions:

jen features 6 different Expression types, as follows:

  • Literals

    Expression Literals are made up of the same 8 data types mentioned above.

  • Function Calls

    jen Functions are called in the form: functionName(parameters)

  • Logical NOT

    Not expression, represented as a bang !, negates any logical expression before it

  • Arithmetic

    Expressions such as add, subtract, multiply, divide, etc

  • Boolean

    The typical AND, and OR are included, as well as the XOR expression

  • Ternary

    jen features a ternary expression, formatted in the same fashion as JavaScript (Conditional ? if true : else)

Statements:

jen utilizes several different statement types, as follows:

  • Loops

    jen features both for, and while loops, using similar syntax to Python

  • Function Declaration

    jen functions are declared using both an annotation, and a signature

  • Declaration

    variables in jen are declared and assigned using different syntax. Variables are declared with an =:

  • Assignment

    variables are assigned using the = sign

  • Conditional

    jen features both the aforementioned Ternary operator, along with the standard if statement. Following similar syntax to Python

  • Return

    naturally, jen allows functions to return a value, using the return keyword
